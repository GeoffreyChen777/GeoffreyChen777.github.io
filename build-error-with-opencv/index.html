gSLIC Build Error.

If you got this error when you build the gSLIC superpixel segmentation algrithm:

/usr/include/c++/5/bits/stl_algobase.h:549:18: error: ambiguous overload for ‘operator=’ (operand types are ‘cv::Rect_’ and ‘cv::Rect_’)

Just add this command in CMakeLists.txt and rebuild them:

set (CMAKE_CXX_STANDARD 11)
